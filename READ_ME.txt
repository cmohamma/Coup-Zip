The zip includes four items:

1) The first dataset is used to train the model and run a 10-fold cross validation and this data ranges from 1960-2010 and is labeled ‘coupriskdata1.csv’. 

2) The second dataset is the full dataset from 1960-2014, which the attached Rscript subsets to post 2010 for the forecasting section of the paper and is labeled ‘pred_data.csv’

3) R script to run cross validation, model selection, forecasting, and mapping. this is labeled ‘coup_script.R’

4) PDF of the article.
